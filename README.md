# Strona warsztatu — AutoFix (przykład)

Pliki:
- `index.html` — główny plik HTML
- `css/styles.css` — style strony
- `js/script.js` — prosta interakcja (menu, walidacja formularza)

Jak uruchomić lokalnie:
1. Otwórz `index.html` w przeglądarce (dwuklik lub "Open with" -> przeglądarka).
2. Aby testować mapę Google, wklej iframe ze swoim adresem w sekcji mapy w `index.html`.

Jak opublikować na Google:
- Google Sites: nie pozwala na bezpośrednie wgranie plików HTML. Jeśli chcesz użyć Google Sites, musisz zbudować stronę w edytorze Sites ręcznie lub osadzić fragmenty (np. HTML) jako ramkę.
- Google Firebase Hosting (zalecane jeśli chcesz statyczny hosting): zainstaluj Firebase CLI, zainicjuj projekt i wypchnij folder z plikami.
